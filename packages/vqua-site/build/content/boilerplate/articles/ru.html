Готовый каркас

Готовый каркас позволит вам быстро приступить к созданию веб-интерфейса с серверным пререндерингом.

Как установить?

{{ install }}

После чего откройте в браузере страницу http://localhost:8080/ и вы должны увидеть страницу Welcome! Test page.

Как редактировать код?

Чтобы изменения вносились сразу после сохранения откройте новое окно треминала и используйте команду

{{ webpack }} Подробнее о webpack

Как редактировать css стили?

Для того чтобы отслеживать изменение стилей в новом окне запустите:

{{ gulp }}

Предпочтительнее всего размещать стили в папках с компонентами. Порядок стилей не должен иметь значения. Для этого вы можете воспользоваться бэм методологией.
